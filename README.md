# proyecto_final_sprint7
Proyecto del sprint 7

El codigo muestra un histograma que describe como se distribuye la cantidad de vehículos por el tipo de vehículo que es.
Adicionalmente también muestra un gráfico de dispersión que muestra como se distribuye el precio de los autos en relación a su kilometraje.
